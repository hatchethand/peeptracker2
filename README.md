# peeptracker2
node.js project for parsing and visualing kismet logs

Example config.json
```
{
	"host": "192.168.1.1",
	"port": 2501,
	"sourceAddress": "192.168.1.2",
	"validMACs":
		[
			"00:00:00:00:00:00",
			"11:11:11:11:11:11"
		],
	"validBSSIDs":
		[
			"22:22:22:22:22:22",
			"33.33.33.33.33.33"
		]
}
```
